# DualFM-Python
## A python module for the DualFM API

### Examples:

#### Song and artist:
```py
import dualfm

print(dualapi.Now.song)
print(dualapi.Now.artist)
```
#### Listeners:
```py
import dualfm

print(dualapi.Listeners.current)
print(dualapi.Listeners.Peak)
```
#### Presenter:
```py
import dualfm

print(dualapi.Presenter.username)
```
